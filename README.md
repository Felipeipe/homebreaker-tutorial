# homebreaker-tutorial
Basic ROS tutorials

# Installation

create a ROS workspace

```bash
mkdir -p tutorial_ws/src
```

Clone this repository into the `src` folder.

```bash
cd tutorial_ws/src
git clone https://github.com/uchile-robotics/homebreaker-tutorial.git
```

Compile.

```bash
cd tutorial_ws
catkin_make
```
